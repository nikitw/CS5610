Week 4 experiments

Getting familiar with AJAX, WebAPI and AngularJS

  • Weather API provided by OpenWeatherMap api
  • AJAX calls using JQuery get calls populate data in Weather Widgets
  • AngularJS basic attributes and controller

I have use OpenWeatherMap's weather api that provides free weather data based on City name and geographical coordinates. The App is self sufficient to demonstrate a lot of features in itself and thus took a good amount of time investment to develop.

I have done AJAX get experiments in Weather APP, and JQuery handles the creation and deletion on widgets from the app page. I had to also handle erroneous data recieved from API.

The AngularJS experiments gives the basic usage of it's attributes and variables.
