This is a music following app with mocked data but a complete navigation functionality using angular-route.js It also implements a hidden detail page that shows details about each artist when clicked throught follow list.

This application demonstrates the angular JS routing and managing different activities on different pages The application allows to add the artist to follow list and then view it on profile page. Profile page also allows to delete the artist from follow list. All these happen without reloading the page. 

					
app.config(['$routeProvider',
  function($routeProvider) {
    $routeProvider.
      when('/home', {
        templateUrl: 'data/002-home.html',
        controller: 'homeCtrl'
      }).
      when('/profile', {
        templateUrl: 'data/002-profile.html',
        controller: 'profileCtrl'
      }).
      when('/details/:artist', {
        templateUrl: 'data/002-details.html',
        controller: 'detailsCtrl'
      }).
      otherwise({
        redirectTo: '/home'
      });
  }]);
