This application demonstrates the angular JS routing and managing different activities on different pages The application allows to add the artist to follow list and then view it on profile page. Profile page also allows to delete the artist from follow list. All these happen without reloading the page. 

					
app.config(['$routeProvider',
  function($routeProvider) {
    $routeProvider.
      when('/home', {
        templateUrl: 'data/004-home.html',
        controller: 'homeCtrl'
      }).
      when('/profile', {
        templateUrl: 'data/004-profile.html',
        controller: 'profileCtrl'
      }).
      when('/details/:artist', {
        templateUrl: 'data/004-details.html',
        controller: 'detailsCtrl'
      }).
      when('/login', {
        templateUrl: 'data/004-login.html',
        controller: 'loginCtrl'
      }).
      when('/user/:username', {
        templateUrl: 'data/004-user.html',
        controller: 'userCtrl'
      }).
      when('/artist/add', {
        templateUrl: 'data/004-artistadd.html',
        controller: 'homeCtrl'
      }).
      otherwise({
        redirectTo: '/home'
      });
  }]);
