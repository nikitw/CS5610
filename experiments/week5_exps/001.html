Home| About| Contact
This is an Angular-routing experiment that handles the navigations on the page through angular-route.js and also allocates individual controllers for each of these pages.

This is Routing controller that controls the url tail which defines which page to navigate when. It navigate to home page when #/home is encountered and to #/about, #/contact respectively.

The below is the code for home controller 

					
		var app = angular.module('PrimeBoxApp', [
		  'ngRoute',
		  'PrimeBoxApp.controllers',
		  'PrimeBoxApp.services'
		]);

		app.config(['$routeProvider',
		  function($routeProvider) {
		    $routeProvider.
		      when('/home', {
		        templateUrl: 'data/000-home.html',
		        controller: 'homeCtrl'
		      }).
		      when('/about', {
		        templateUrl: 'data/000-about.html',
		        controller: 'aboutCtrl'
		      }).
			  when('/contact', {
		        templateUrl: 'data/000-contact.html',
		        controller: 'contactCtrl'
		      }).
		      otherwise({
		        redirectTo: '/home'
		      });
		  }]);
