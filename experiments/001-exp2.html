CSS experiments

Rotated div

CSS can be used to rotate a div using transform:rotate(#deg) attribute

this box should be rotated 30 deg

            	.rotate {
			    width: 100px;
			    height: 100px;
			    background-color: #f8daa2;
			    -ms-transform: rotate(30deg);
			    -webkit-transform: rotate(30deg);
			    transform: rotate(30deg);
			}

Skewed div

A div can be skewed using transform:skew(#deg) attribute

this box should be skewed 30 deg

            	.skewed {
			    width: 100px;
			    height: 100px;
			    background-color: #f8daa2;
			    -ms-transform: skew(30deg,20deg); /* IE 9 */
			    -webkit-transform: skew(30deg,20deg);
			    transform: skew(30deg,20deg);
			}

Width transition div

Transitions can be provided to edges of the div using transition: width #s height #s transform #s and :hover in combination

hover here to see width transitions

            	.width-trans {
			    height: 100px;
			    -webkit-transition: width 2s;
			    transition: width 2s, height 2s, transform 2s;
			}
			    .width-trans {
			    height: 100px;
			    -webkit-transition: width 2s;
			    transition: width 2s, height 2s, transform 2s;
			}

Resizable div

A div can be made resizable so that user can adjust its edges simply by dragging it in/out. resize:both

this is a resizable div

 			.resize {
			    width: 400px;
			    resize: both;
			    overflow: auto;
			}
