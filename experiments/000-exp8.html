Home page

This is a parallax page where the background will scroll along with the scrolling page but at different speeds such that offering a smooth transition.

We need...

  • An attractive image
  • CSS3 background property
  • CSS3 transparency property
  • Little Jquery using Google api

About page

the below code is used for parallax effect


                $(window).scroll(function(e){
                  parallax();
                });
                function parallax(){
                  var scrolled = $(window).scrollTop();
                  $('.bg').css('top',-(scrolled*0.2)+'px');
                }
