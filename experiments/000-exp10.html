Smooth Scrolling

The feature you must have observed when the page loaded when you navigated to the URL. It silently sweeps window scroll to 30 (look at the scroll debug value at left fixed <div>) The effect is also seen when you scroll the page to its top extreme, it resets the window scroll to 30 smoothly. If you have a touch screen please scroll a finger down to see the effect.

$(window).scroll($.debounce(0, true, function (e) { smoothscroll(); })); $(window).scroll($.debounce(250, function (e){ var scrolled = $(window).scrollTop(); if (scrolled < 30) smoothScroll(30, 'slow'); })); function smoothScroll(top, speed) { $('html,body').animate({ scrollTop: top }, speed); }
