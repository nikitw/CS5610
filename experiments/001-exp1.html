HTML5 features

HTML5 has various inbuilt features that ease up the website building hence further.

<a> anchor tag

Links and references to other pages on the same site or to an altogether different domain can be done via Anchor a tags. There are also various target locations configurable for each anchor.

Google

<img> image tag

Images can be put on a web page using img tag just by specifying it's source URL

Image maps

Image Maps is a unique feature of HTML5 which allows us to target multiple selective areas on a single image, each when clicked, behaves differently as per it's specified redirection.

Figure

Figure tag is can nest the image with it's specific figure caption. Caption is placed right below the image once loaded.

Editable To-Do list. Click one of the item.

This feature helps generation/modification of a TO-DO list on a webpage. It is a pure HTML5 feature, no CSS implied.

Input type email

This imput type enables valid email address verification by the page itself without using any javascripts or server side validations.

Placeholders

Placeholders occupy the space on an input box unless focused by a User.

Header and Footer Tags

Audio support

HTML5 has an inbuilt audio support that can play audio files of MP3, OGG etc. formats.

Video support

Video playback support is now a part of HTML5 thus replacing most of Flash content on web. It is a fast secure and reliable playback feature by HTML5. All the media playback codecs are supported by most of the HTML5 supported browsers.

Range

HTML5 provides a gauge feature that can span up to certain specified range and can be used as a soft tool with the help of a tiny CSS and Java Script.
