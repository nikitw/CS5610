About

The application fetches data from a NodeJS app server that it resides in. It performs a server side authentication. If the credentials are correct then it returns the user and routes to the homepage where it list all the artists on server. App is now capable of registering a new user.

app.js

Application JS runs on NodeJS server as \>$ node app.js listening to port# 8080. It serves Get requests to server. 

          
		app.post('/app/register', function (req, res) {
	    var username = req.body.username;
	    var auser = null;
	    var user = new User(
	      req.body.name,
	      username,
	      req.body.password,
	      req.body.hint,
	      req.body.admin
	    );
	    for (var u in users)
	      if(users[u].username == username) {
	        auser = users[u];
	        break;
	      }
	    if(auser) {
	      user.err = 702;
	      user.errmsg = "user already exists";
	    } else {
	      users.push(user);
	    }
	    res.jsonp(user);
	});

Service ajax

Service request post fetches data from the /app/register REST API and callback the requesting method and routes to homepage when login successful or displays an error if the username already exists.

          
	services.factory('userService', function($http) {
	    ...

		us.register = function (user) {
	        $http.post(URL+"register", user)
	          .success(function(auser) {
	            if(!auser.err) {
	              user.success(auser);
	              us.currUser = auser;
	            } else
	              user.error(new Error(auser.err, auser.errmsg));
	          });
	    };

	    ...

	    return us;
	  });
