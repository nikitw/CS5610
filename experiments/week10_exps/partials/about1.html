About

Google's Map API allows to diplay the map and location on the browser with less effort. This experiments demonstrates the application of maps on HTML5 canvas with functionality that can have various feature based advantages. The default location is Boston.

Client JS Code

The API call places map object that initializes the canvas over a div with a specified ID on window.load. 

        
            function initialize(e, loc, zoom) {
            if(loc == null || loc == undefined)
            loc = {lat: 42.333821, lng: -71.0966081};
            var mapOptions = {
            center: loc,
            zoom: (zoom)? zoom: 8
            };
            var map = new google.maps.Map(document.getElementById('map-canvas'),
            mapOptions);

            var marker = new google.maps.Marker({
            position: loc,
            map: map,
            title: $('#address').val()
            });

            return loc;
            }
