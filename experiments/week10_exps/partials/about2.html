About

Google's Map API allows to diplay the map and location on the browser with less effort. This experiments demonstrates the HTML5 geolocation feature (supported browsers only). It asks for user to allow location access for this page and grabs the location from browser to display the map with the marker with maximum possible accuracy.

Client JS Code

The API call places map object that initializes the canvas over a div with a specified ID on window.load. 

        
            function getLocation(callback) {
            if (navigator.geolocation) {
            navigator.geolocation.getCurrentPosition(callback);
            } else {
            callback({ err: "Geolocation is not supported by this browser."});
            }
            }
