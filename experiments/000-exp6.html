This is a simple example on how to use the previously created grid view to generate a neat layout for an online book reader. The header specifies the menu buttons generated using 3 cells with 2 column width and a search menu with 6 column width cell. The navigation spans 3 column width cells, while the center page spans remaining 8 column width cells. The Footer has page number column spanned over 10 cells and the remaining 2 with the next and previous page function spanning 1 cell each.
