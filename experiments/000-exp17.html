About

The experiments are to demonstrate JQuery UI features that allow dragging sorting and resizing of dom elements The next experiemnt tab represents the feature that stores the data on browser memory as a database using localStorage. Whenever the save button is pressed all the paragraphs created using Add Paragraph buttons gets saved to the memory and when we reload, those paragraphs are loaded from the memory and rendered on their respective position.

function addPara( text, x, y) {
	var p = $("<p contenteditable=true>");
	p.addClass("para");
	p.draggable({
		grid: [20,20]
	});
	p.css({
		position: 'absolute',
		left: x,
		top:  y
	});
	p.html(text);
	$("#2").append(p);
}
