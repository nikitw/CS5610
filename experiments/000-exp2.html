CSS experiments

Rounded border

Rounded borders can be generated using CSS as shown below using attribute border-radius:4px;

this is a rounded paragraph!


            	.rounded {
			    border: 2px solid;
			    padding: 10px;
			    border-radius: 25px;
			}

Box Shadow

CSS3 can generate shadows for any div or p using attribute box-shadow

this box will have a shadow

            	.box-shadow {
			    height:100px;
			    margin: 30px;
			    box-shadow: 10px 10px 5px #757575;
			}

Gradient

Gradients can be applied to the backgrounds of dom elements using background: linear-gradient(#c, #c), radial, skewed and angular gradients can also be provided using CSS

this is a simple gradient

            	.gradient {
			    background: -webkit-linear-gradient(yellow, orange); 
			    background: -o-linear-gradient(yellow, orange); 
			    background: -moz-linear-gradient(yellow, orange); 
			    background: linear-gradient(yellow, orange); 
			}

Text shadow

CSS can provide shadows to texts using text-shadow

this is text with shadow

            	.text-shadow {
			    text-shadow: 5px 5px 5px #757575;
			}
