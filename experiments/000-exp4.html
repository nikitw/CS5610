About

This experiments explains the liquid design, technically called responsive website that can adjust the layout based on the screen size. The above divs are tiles in a desktop view and they stack when the screen is shortened widthwise. The Navigation is also the same responsive which is implemented on the home page and it stacks in to dropdowns as soon as the screen starts getting small.

Code


			<div class="tile left">
            			<h1>Tile 1</h1>
	        	</div>
	        	<div class="tile left">
            			<h1>Tile 2</h1>
        		</div>
